Home

Hallo und willkommen zum DukeKiosk webinterface! In der Zukunft sind hier Informationen über bisherige Kundenaktivitäten angedacht. Aktuell gibt es eine kurze Einführung darüber, was auf welcher Seite zu finden ist.

Administratoransichten

Produkte

Erstelle oder bearbeite Produkte. Verwalte Preise, Barcodes, usw.

Kunden

Erstell oder bearbeite Benutzer, Verawalte Namen, ID barcodes, usw.

Mitarbeiteransichten

Verkaufsstation

Diese Ansicht kann von einem Mitarbeiter genutzt werden, um Kunden-checkouts zu verwalten. Sollte mit einem barcode scanner verwendet werden.

Kassenstation

This view can be used by a kiosk employee to handle customer payments. Sollte mit einem barcode scanner verwendet werden.

Kundenansichten

Selbstbedienungsstation

Diese Ansicht kann auf einem kiosk terminal angezeigt werden. Kunden können hier ihre Produkte selbstständig eingeben. Diese Ansicht hat eine begrenzte Funktionalität und ein anderes Nutzerinterface. Sollte mit einem barcode scanner verwendet werden.
